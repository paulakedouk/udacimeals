# Udaci-meals

This repo is for [Udacity](https://www.udacity.com/)'s Front End Frameworks course. 

## Required Software

* [Git](https://git-scm.com/)
* [npm](http://npmjs.com/)

## Getting Set Up

* clone this repo
* `cd` into the newly cloned project
* install npm dependencies via `npm install`

## View the app

* Run `npm start` and :boom:
